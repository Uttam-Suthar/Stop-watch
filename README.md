<h2>Features</h2>

<ul>
    <li>
        <h3>Start timer</h3>
        <ul>
            <li>On click of start button, start the timer</li>
            <li> â—‹ If timer is 0 start from beginning else start from wherever last stopped
            </li>
        </ul>

    <ul>
        <li>
            Stop timer once clicked on the stop button
